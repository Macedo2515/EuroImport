# Euroimport Motors

## Descrição
Sistema web de gestão de importação de veículos premium europeus.
Permite aos clientes solicitar importações, acompanhar todo o processo
negocial e efetuar pagamentos. A equipa interna gere propostas,
documentação, inventário e aprovações por níveis hierárquicos.

## Funcionalidades
- Página institucional com catálogo de veículos, serviços e contactos
- Simulador de ISV
- Pedido de proposta online
- Formulário de contacto com envio de email
- Registo, login e recuperação de password por email
- Dashboards personalizados por perfil:
  - **Cliente** — submeter e acompanhar pedidos
  - **Funcionário** — gerir negócios e inventário
  - **Administrador** — aprovar veículos e supervisionar processos
  - **Direção** — aprovação final de decisões
- Gestão completa do ciclo de negócio (submissão → proposta →
  renegociação → pagamento → importação → conclusão)
- Upload e validação de documentos
- Sistema de notificações em tempo real
- Gestão de inventário com aprovação hierárquica

## Como instalar e correr
1. npm install
2. node server.js
3. Abrir http://localhost:3000 no browser

## Tecnologias usadas
- Node.js + Express
- HTML / CSS / JavaScript
- MySQL
- Nodemailer (envio de emails)
- Bcrypt (segurança de passwords)
- Express-Session (autenticação)
- Multer (upload de ficheiros)
