# Euroimport Motors

Description
Web management system for premium European vehicle imports. Allows clients to request imports, track the entire negotiation process and make payments. The internal team manages proposals, documentation, inventory and hierarchical approvals.

Features

Institutional website with vehicle catalogue, services and contacts
ISV tax simulator
Online proposal request
Contact form with email sending
Registration, login and password recovery via email
Personalised dashboards by profile:

Client — submit and track requests
Employee — manage businesses and inventory
Administrator — approve vehicles and supervise processes
Management — final approval of decisions


Full business cycle management (submission → proposal → renegotiation → payment → import → conclusion)
Document upload and validation
Real-time notification system
Inventory management with hierarchical approval


Getting Started
bashnpm install
node server.js
Open http://localhost:3000 in your browser.

Tech Stack

Node.js + Express — backend framework
HTML / CSS / JavaScript — frontend
MySQL — database
Nodemailer — email sending
Bcrypt — password security
Express-Session — authentication
Multer — file uploads
